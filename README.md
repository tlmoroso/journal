# journal

Personal Website for Journaling.

## Tech Stack

- [Tanstack Start](https://tanstack.com/start) (Vite + Nitro/h3)
- [Tanstack Router](https://tanstack.com/router), [Tanstack Query](https://tanstack.com/query), [Tanstack Table](https://tanstack.com/table)
- [Zod 4](https://zod.dev)
- TypeScript (strict), ESLint 9, Prettier, Vitest
- pnpm with [catalog](https://pnpm.io/catalogs) for centralized version management

## Prerequisites

- Node **24** (LTS). Use [nvm](https://github.com/nvm-sh/nvm) and run `nvm use` (`.nvmrc` pins the version).
- pnpm **9.5+**. Easiest path: `corepack enable` (Node ships Corepack; `package.json#packageManager` pins the exact pnpm version).

## Getting Started

```bash
corepack enable          # activates the pinned pnpm version
nvm use                  # reads .nvmrc -> Node 24
pnpm install             # resolves catalog references
pnpm dev                 # http://localhost:3000
```

## Scripts

| Script              | Description                                                |
| ------------------- | ---------------------------------------------------------- |
| `pnpm dev`          | Start dev server (Vite + Nitro/h3 in one process)          |
| `pnpm build`        | Production build (output in `dist/`)                       |
| `pnpm start`        | Run the production build                                   |
| `pnpm typecheck`    | Run `tsc --noEmit`                                         |
| `pnpm lint`         | Run ESLint                                                 |
| `pnpm lint:fix`     | Run ESLint with `--fix`                                    |
| `pnpm format`       | Run Prettier `--write`                                     |
| `pnpm format:check` | Run Prettier `--check`                                     |
| `pnpm test`         | Run Vitest once                                            |
| `pnpm test:watch`   | Run Vitest in watch mode                                   |
| `pnpm test:ui`      | Open the Vitest UI                                         |
| `pnpm check`        | Run typecheck + lint + format:check + test (CI-equivalent) |
| `pnpm clean`        | Remove build artifacts                                     |

## Project Structure

```text
├── styles.css
├── components/                 # shared UI
├── lib/
│   ├── query-client.ts         # QueryClient factory
│   └── ...                     # shared data/query helpers
├── server/
│   └── ...                     # server-only functions
└── routes/
    └── __root.tsx, index.tsx   # pages
```

## Dependency Versions

All versions are managed in a single `default` catalog in `pnpm-workspace.yaml`. To bump a dependency, edit the version range there and run `pnpm install`.
